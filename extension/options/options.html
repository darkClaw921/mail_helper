Extension settings

Backend connection

The HTTP origin of your Mail Helper server. WebSocket URL is derived automatically by swapping http:// → ws:// (https → wss).

Generated on first backend start and stored in settings.api_key. Used for both X-API-Key header and WS ?token= query.

Notifications

Uses chrome.notifications. You may also need to enable browser-level notifications in OS settings.
